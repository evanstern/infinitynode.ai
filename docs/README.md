# Docs

This folder is the durable memory for how this homelab assistant is built and operated.

- `workflow.md` — how we plan work (Forgejo Issues) and where design/runbooks live.
- `plans/` — design notes and phase plans (links out to Forgejo issues).
- `runbooks/` — step-by-step operational guides (how to run/restore/debug).
- `adr/` — lightweight Architecture Decision Records (optional, used when a decision matters).
